

# План: Карточки с фото в блоке «Что мы предлагаем»

## Что будет сделано

Карточки в секции «Направления» получат фотографии с эффектом перехода из черно-белого в цветной при наведении.

## Визуальная структура карточки

```text
┌─────────────────────────────┐
│                             │
│      [Фото]                 │  ← Ч/б по умолчанию
│      grayscale → color      │  ← Цветное при наведении
│                             │
├─────────────────────────────┤
│  Заголовок                  │
│  Описание...                │
└─────────────────────────────┘
```

## Технические детали

### 1. Загрузка фотографий
Вам нужно будет загрузить 4 цветных фотографии в чат после одобрения плана:
- Фото для «Оборудование»
- Фото для «Запасные части»
- Фото для «Расходные материалы»
- Фото для «Сервисный инжиниринг»

### 2. CSS-эффект grayscale
Черно-белый эффект создается через CSS-фильтр:
```css
/* По умолчанию — черно-белое */
filter: grayscale(100%);
transition: filter 0.5s ease;

/* При наведении — цветное */
&:hover {
  filter: grayscale(0%);
}
```

### 3. Изменения в Directions.tsx
- Добавить поле `image` в массив `directions`
- Убрать иконки (Settings, Cpu, Droplets, Wrench)
- Добавить контейнер для фото с overflow: hidden
- Применить CSS-фильтр и плавную анимацию
- Сохранить заголовки и описания под фото

### 4. Структура карточки
- Фото занимает верхнюю часть карточки (aspect-ratio 4:3 или 16:9)
- Заголовок и описание располагаются под фото
- При наведении на всю карточку — фото становится цветным
